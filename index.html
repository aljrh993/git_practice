  • Using basic git commands




  • git init - initializes a local git repository in the folder
  • git add . - add all the new files in staging
  • git status
  • git commit -m "__message__" - commits the changes
  • git remote add origin __github_url___ - adds as a remote place called 'origin' that points to your github_url
  • git push - pushes your latest changes to github
  • git clone __github_url__ - clones a github repository to your local working directory
